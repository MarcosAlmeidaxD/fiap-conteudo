# Cap 12 - Modelos supervisionados e não supervisionados RevFinal.pdf

**File type:** PDF | **Processed:** 2026-04-22

## TL;DR
A foundational guide to Supervised and Unsupervised Learning that treats Machine Learning as a way to "discover" programs through data rather than writing them manually.

## What's Inside
- **Concept Mapping:** Venn diagrams showing the messy overlap between AI, Machine Learning, Data Science, and KDD.
- **Real-World Case Studies:** Breakdowns of Google’s Page Rank, collaborative filtering for recommendations, and automated translations.
- **Optimization & Evaluation:** Practical sections on Loss Functions, Overfitting (explained via memes), and model evaluation metrics.
- **Python Snippets:** Ready-to-use source code for implementing SVM with GridSearch and fine-tuning models using the Pycaret library.
- **Training Drills:** Specific activities to test your knowledge on evaluation metrics and identifying "overfit" scenarios.

## Worth Knowing
The guide highlights a paradigm shift: in traditional programming, you provide the rules; in Machine Learning, you provide the data and the results, and the machine *returns* the rules (the function). It also notes that most Big Data is currently wasted because humans are traditionally bad at defining functions for massive datasets.
